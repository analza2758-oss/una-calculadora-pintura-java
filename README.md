## 📌 Descrição do Projeto

Este projeto foi desenvolvido como atividade da disciplina de lógica de programação, com o objetivo de criar um sistema em Java capaz de calcular a quantidade de tinta necessária para pintar um quarto.

O programa solicita ao usuário a medida do lado de um quarto quadrado e, com base nisso, realiza os cálculos necessários para determinar a área total das paredes e a quantidade de tinta necessária.

---

## ⚙️ Funcionalidades

* Solicita ao usuário a medida do lado do quarto (em metros)
* Considera altura padrão de 2,70 metro
* Calcula a área total das quatro paredes
* Aplica desconto de 10% referente a portas e janelas
* Calcula a quantidade de tinta necessária
* Exibe o resultado final em litros

---

## 🧮 Lógica Utilizada

1. Cálculo da área de uma parede:
   `lado × altura`

2. Cálculo da área total das quatro paredes:
   `4 × (lado × altura)`

3. Aplicação do desconto de 10%:
   `área total × 0,9`

4. Cálculo da quantidade de tinta:
   `área final ÷ 10`
   (considerando que 1 litro cobre 10 m²)
